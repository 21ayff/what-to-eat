# 🍽️ 今天吃什么 - 智能美食推荐

一个功能丰富的美食推荐应用，帮助解决"今天吃什么"的世界难题！

## ✨ 功能特点

- 🧠 **智能推荐** - 基于7个维度（时段、心情、天气、人数、口味、预算、健康）智能推荐
- 🎮 **趣味游戏** - 美食转盘、骰子挑战、快问快答
- 📚 **美食图鉴** - 216道美食，涵盖中外各大菜系
- 👤 **个人中心** - 记录探索历程，收集成就徽章

## 🎨 界面预览

- 温暖橙色调设计，充满食欲感
- 流畅动画效果
- 响应式布局，支持手机/平板/电脑

## 🚀 技术栈

- Vue 3 + Vite
- 纯前端实现，无需后端

## 📦 部署

### 本地开发
```bash
npm install
npm run dev
```

### 生产构建
```bash
npm run build
```

### Vercel 部署
1. 推送到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成

## 📝 美食分类

- 中式（川菜、粤菜、湘菜、鲁菜等八大菜系）
- 西式、美式、意大利
- 日式、韩式
- 印度、法国、地中海、墨西哥
- 东南亚
- 健康轻食
